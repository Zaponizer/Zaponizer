# Neon site (HTML/CSS/JS)

## Запуск

Любой статический сервер. Например:

```bash
cd site
python -m http.server 5173
```

Откройте `http://localhost:5173/`.

## Что уже реализовано

- Фиксированная верхняя панель: **О создателе**, **Вход/Профиль**, **RU/EN**.
- Переключатель разделов (слайдер-кнопки): **Zaponizer** / **The Soul's Contour**.
- Главная: черный фон + “пепел” (частицы) + 2 колонки неоновых кнопок, которые меняются от слайда.
- Рандомные градиенты при каждом обновлении.
- Модальное окно входа/регистрации.
- Очки/статистика в профиле (хранение в браузере через `localStorage`).
- “Секретник”: ввод кода и начисление очков.

## Важно

Регистрация сделана **локально** (в браузере) через `localStorage` — это не “настоящая” безопасность.
Если нужен реальный аккаунт/серверная БД/админка/таблица лидеров — скажите, добавлю backend.
